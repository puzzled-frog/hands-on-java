# Requirements: Hello Calculator

## Overview
Build a command-line calculator that performs basic arithmetic operations on two numbers.

## Functional Requirements

### Input
- Prompt the user to enter the first number
- Prompt the user to enter an operator (+, -, *, /)
- Prompt the user to enter the second number
- Read and process user input from the console

### Calculation
- Perform addition when the operator is +
- Perform subtraction when the operator is -
- Perform multiplication when the operator is *
- Perform division when the operator is /

### Output
- Display the calculation result

## Non-Functional Requirements

### Usability
- Prompts should be clear and easy to understand
